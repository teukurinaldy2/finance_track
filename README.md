# finance_track
Aplikasi pencatatan keuangan sederhana dibangun menggunakan python dan django. Pengguna dapat mencatat pemasukan dan pengeluaran serta memantau saldo

# fitur
- Pencatatan keuangan pemasukan dan pengeluaran
- Managemen katergori transaksi
- laporan keuangan bulanan

# Tech Stack
- Python
- Django
- MYSQL
- HTML
